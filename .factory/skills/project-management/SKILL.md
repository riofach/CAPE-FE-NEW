# Project Management Protocol

Tugasmu adalah memperbarui file `.factory/STATUS.md`.

## Actions:

1.  **Mark Complete:** Jika user bilang fitur selesai, centang [x] di `STATUS.md`.
2.  **Add Task:** Jika ada ide baru, masukkan ke daftar Backlog.
3.  **Reflect:** Baca `STATUS.md` dan beri saran ke user apa yang harus dikerjakan selanjutnya ("Next Step").

JANGAN mengubah kode program saat skill ini aktif. Fokus hanya pada administrasi project.
